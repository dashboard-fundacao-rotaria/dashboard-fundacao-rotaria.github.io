Guia de Utilização

Como navegar neste relatório

Use o menu no topo da página para alternar entre as seções: contribuição por distrito, contribuição por fundo e o comparativo entre metas e resultados. Basta clicar no nome da seção desejada.

Em cada página, os filtros permitem focar apenas no que você quer ver. Clique no filtro de distrito e selecione o seu — todos os gráficos da página se ajustam automaticamente ao que foi escolhido. Para voltar a ver todos os distritos, é só limpar a seleção.

Para ver valores exatos, passe o cursor sobre as barras, fatias ou pontos dos gráficos: uma caixa aparece com os números detalhados daquele item.

Algumas seções têm a indicação "Atualizado em:", mostrando a data dos dados mais recentes daquela área.

Se a tela estiver pequena, use o modo de tela cheia (ícone no canto superior) para uma visualização mais confortável.
